# S3 Storage Settings

Configure S3-compatible storage for server backups. Supports any S3-compatible provider (AWS S3, Backblaze B2, Cloudflare R2, etc.).

## Get S3 Settings

```http
GET /api/v1/settings/s3
```

Retrieves the current S3 storage configuration (secrets are masked).

### Example Response
```json
{
  "data": {
    "endpoint": "https://s3.us-east-1.amazonaws.com",
    "bucket": "my-escluse-backups",
    "region": "us-east-1",
    "access_key_id": "AKIA***WXYZ",
    "configured": true
  }
}
```

## Update S3 Settings

```http
PUT /api/v1/settings/s3
```

Updates S3 storage configuration.

### Request Body
```json
{
  "endpoint": "https://s3.us-east-1.amazonaws.com",
  "bucket": "my-escluse-backups",
  "region": "us-east-1",
  "access_key_id": "AKIAIOSFODNN7EXAMPLE",
  "secret_access_key": "wJalrXUtnFEMI/K7MDENG/bPxRfiCYEXAMPLEKEY"
}
```

### Example Request

::: code-group

```bash [curl]
curl -X PUT https://api.esluce.com/api/v1/settings/s3 \
  -H "Authorization: Bearer ${ESCLUSE_API_KEY}" \
  -H "Content-Type: application/json" \
  -d '{"endpoint": "https://s3.us-east-1.amazonaws.com", "bucket": "my-escluse-backups", "region": "us-east-1", "access_key_id": "AKIAIOSFODNN7EXAMPLE", "secret_access_key": "wJalrXUtnFEMI/K7MDENG/bPxRfiCYEXAMPLEKEY"}'
```

```typescript [Node.js SDK]
await client.settings.updateS3({
  endpoint: 'https://s3.us-east-1.amazonaws.com',
  bucket: 'my-escluse-backups',
  region: 'us-east-1',
  accessKeyId: 'AKIAIOSFODNN7EXAMPLE',
  secretAccessKey: 'wJalrXUtnFEMI/K7MDENG/bPxRfiCYEXAMPLEKEY'
});
```

```python [Python SDK]
client.settings.update_s3(
  endpoint='https://s3.us-east-1.amazonaws.com',
  bucket='my-escluse-backups',
  region='us-east-1',
  access_key_id='AKIAIOSFODNN7EXAMPLE',
  secret_access_key='wJalrXUtnFEMI/K7MDENG/bPxRfiCYEXAMPLEKEY'
)
```

:::

### Possible Errors
| HTTP | Code | Description |
|------|------|-------------|
| 400 | `VAL_INVALID_FORMAT` | Invalid S3 endpoint or credentials format |
| 403 | `GEN_FORBIDDEN` | Insufficient permissions |
