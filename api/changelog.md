# API Changelog

This document tracks significant changes, additions, and deprecations to the Escluse API.

## 2026

### May 2026

- **Phase 52 — API Documentation Launch**: Comprehensive API documentation now available at `docs.esluce.com/api/`. Includes full endpoint reference, authentication guide, error code catalog, and SDK quickstarts.
- **New endpoint groups documented**: Settings (S3, Cloudflare), Templates (server, plugins, modpacks), Agents, Profiling, Git operations, Build system, Deploy API, Cron tasks, Alerts, Webhooks.

### April 2026

- Initial API v1 release with support for Servers, Nodes, Billing, and basic authentication.
